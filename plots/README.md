## Data files

File `31eval-Q100.txt` and `32eval-old.txt`:

1. Caller name
2. LCR truth
3. LCR FN
4. SegDup truth
5. SegDup FN
6. Rest truth
7. Rest FN
8. LCR calls
9. LCR FP
10. SegDup calls
11. SegDup FP
12. Rest calls
13. Rest FP

File `33len-Q100.txt`:

* Col 1: caller name
* Col 4,6,8,10,12,14: truth counts for 2k+, 1-2k, 500-1k, 200-500, 100-200 and 50-100, respectively
* Col 5,7,9,11,13,15: corresponding FN
* Col 18,20,22,24,26,28: number of calls for 2k+, 1-2k, 500-1k, 200-500, 100-200 and 50-100, respectively
* Col 19,21,23,25,27,29: corresponding FP

File `23len.txt`: region lengths
